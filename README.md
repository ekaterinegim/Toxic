# Toxic
Создала сайт отеля с использованием Pug, Sass и Webpack. Проект включает структурированную верстку, стилизацию, автоматическую сборку ресурсов, обеспечивая удобство бронирования и современный дизайн.
Этот проект — адаптивный сайт отеля, созданный с использованием Webpack, Pug, Sass и jQuery.

## Установка и запуск

1. **Клонируйте репозиторий:**

```bash
git clone [https://github.com/ваш_никнейм/имя_репозитория](https://github.com/ekaterinegim/Toxic).git
```

2. **Перейдите в папку проекта:**

```bash
cd имя_проекта
```

3. **Установите зависимости:**

```bash
npm install
```

4. **Запустите проект в режиме разработки:**

```bash
npm run dev
```

После этого проект откроется в браузере по адресу `http://localhost:8080`.

## Структура проекта

- `src/` — исходные файлы (Pug, Sass, JS)
- `dist/` — собранные файлы для публикации

## Команды npm

- `npm run dev` — запуск в режиме разработки с автоматической перезагрузкой
- `npm run build` — сборка проекта для продакшена

## Примечание

Перед началом убедитесь, что у вас установлены Node.js и npm.
